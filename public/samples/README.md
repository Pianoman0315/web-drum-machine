Place the fixed drum samples in this folder with these exact names.
Version 2.1 currently also uses placeholder mappings for `Accent`, `Tom 1`, and `Tom 2`
until dedicated samples are assigned in code.

- `bass.wav`
- `snare.wav`
- `clap.wav`
- `chihat.wav`
- `ohat.wav`
- `cow.wav`

The app loads them from `/samples/*.wav` through Tone.js.
